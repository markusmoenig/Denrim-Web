# Denrim: Forge

**Easy 3D creation for game props, figurines, and printable assets.**

<p className={styles.platformLine}>Native for <strong>macOS and iPadOS</strong>, with iCloud sync.</p>

Denrim Forge is a focused low-poly modeler for artists who want to make game-ready props, stylized characters, and assets for 3D printing without getting lost in a complex toolset. Build directly in the viewport, finish with paint and materials, then export to the workflow you need.

## Main Features

- **Beginner-friendly modeling** - Start with primitives and shape them using direct tools for transforms, push/pull, extrude, inset, bevel, cut, lathe, and more.
- **Game props and figurines** - Make crates, buildings, vehicles, stylized characters, and printable forms with a clean low-poly workflow.
- **Integrated help everywhere** - Contextual guidance explains each tool, mode, workflow, and constraint while you work.
- **Mesh tools and symmetry** - Select, repair, mirror, subdivide, bridge, weld, and refine topology without leaving the editor.
- **Materials, paint, and UVs** - Apply palette materials, paint face-level details, and adjust UVs directly in the viewport.
- **Render and inspect** - Switch to a clean render view with lighting, reflections, tone mapping, floor, and presentation controls.
- **Flexible export** - Export OBJ texture bundles, vertex-color OBJ, GLB, USDZ, STL, and PNG renderings.
- **Cross-platform workflow** - Native apps for macOS and iPadOS, with document-based projects and iCloud-backed project library support.

Create a simple blockout, refine its silhouette, add details, and take it into a game engine, a renderer, or a 3D-printing workflow. Forge keeps the essential steps close together so you can learn by making.

**Available soon for macOS and iPadOS.**

# Denrim: Noise

**Professional procedural noise generation for artists, designers, and developers.**

<p className={styles.platformLine}>Native for <strong>macOS and iPadOS</strong>, with iCloud sync.</p>

Create stunning textures, backgrounds, and patterns with advanced noise algorithms. Denrim Noise combines powerful procedural generation with an intuitive interface, making it easy to craft unique visuals for games, digital art, web design, and more.

## Main Features

- **7 Noise Algorithms** - OpenSimplex (2S), Perlin, Value, Cellular, Sinusoidal, and Gabor noise types
- **Advanced Fractal System** - Layer up to 8 octaves with FBM, Ridged, and Ping Pong fractals
- **Seamless Tiling** - Generate perfectly tileable patterns for games and 3D textures
- **Domain Warping** - Create organic, flowing patterns with dual-layer warp controls
- **Gradient Coloring** - Transform grayscale noise with customizable 4-color gradients
- **3D Rendering** - Realistic bump mapping with adjustable lighting and material properties
- **Flexible Export** - Save at any resolution (512px-8192px) in multiple aspect ratios and formats
- **Cross-platform** - Native apps for macOS and iPadOS with iCloud sync

Perfect for creating terrain heightmaps, seamless textures, desktop wallpapers, game assets, and abstract art. Start generating beautiful procedural patterns in minutes.

**Available now for macOS and iPadOS as well as Windows and Linux.**
