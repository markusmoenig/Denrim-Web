**Professional procedural noise generation for artists, designers, and developers.**

Create stunning textures, backgrounds, and patterns with advanced noise algorithms. Denrim Noise combines powerful procedural generation with an intuitive interface, making it easy to craft unique visuals for games, digital art, web design, and more.

## Main Features

- **7 Noise Algorithms** - OpenSimplex (2S), Perlin, Value, Cellular, Sinusoidal, and Gabor noise types
- **Advanced Fractal System** - Layer up to 8 octaves with FBM, Ridged, and Ping Pong fractals
- **Seamless Tiling** - Generate perfectly tileable patterns for games and 3D textures
- **Domain Warping** - Create organic, flowing patterns with dual-layer warp controls
- **Gradient Coloring** - Transform grayscale noise with customizable 4-color gradients
- **3D Rendering** - Realistic bump mapping with adjustable lighting and material properties
- **Flexible Export** - Save at any resolution (512px-8192px) in multiple aspect ratios and formats
- **Cross-platform**: Native apps for macOS and iPad with iCloud sync

Perfect for creating terrain heightmaps, seamless textures, desktop wallpapers, game assets, and abstract art. Start generating beautiful procedural patterns in minutes.
